### my-youtube-app  

### How to setup  
    1. Clone the repository  
    2. `cd my-youtube-app`    
    3.  `npm install`    
    4. `npm run start`    

### Tech stack  
    1. ReactJS  
    2. TailwindCSS  
    3. Redux  
    4. React-router  

### Modules  
    1. Video Listing  
    2. Video Detail  
    3. Debounced search  

Create an .env file and place your Youtube API key as REACT_APP_GOOGLE_API_KEY which can be accessed in constants file.
